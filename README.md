🎓 Travaux BUT MMI

Fichiers réalisés dans le cadre de ma formation Bachelor Universitaire de Technologie — Métiers du Multimédia et de l'Internet (BUT MMI).


📚 Contexte :

Ce dépôt regroupe les projets, exercices et rendus produits durant mes années de BUT MMI. Ces travaux ont été réalisés à des fins pédagogiques, dans le cadre de mes cours et de mes SAÉ (Situations d'Apprentissage et d'Évaluation).

🗂️ Contenu :

Les fichiers présents dans ce dépôt sont mes réalisation des SAÉ centré sur le développemnt web: 
  - Le HTML
  - Le CSS
  - Le PHP
  - Le TWIG


⚠️ Avertissement :

Ces fichiers sont mes travaux d'étudiant de début de formation. Ils peuvent contenir des erreurs, des expérimentations et/ou du code non optimisé — c'est tout à fait normal, c'est l'essence même de l'apprentissage. 🙂

▶️ Visualiser les projets :

Pour afficher correctement certains rendus (notamment en PHP ou TWIG), il est nécessaire d’utiliser un serveur local, par exemple :
  - XAMPP (gratuit)
  - WAMP
  - MAMP

Il suffit de placer les fichiers dans le dossier htdocs (ou équivalent) puis d’accéder au projet via localhost.
